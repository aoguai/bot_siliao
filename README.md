# bot_siliao
基于botoy和OPQBot的2022私聊转发插件
## # [私聊转发系统(bot_siliao)](https://github.com/opq-osc/bot_siliao)
支持：<br>
	黑名单、指定关键词不转发<br>
	图片消息、文字消息、图文消息转发<br>
	图片消息、文字消息、图文消息回复<br>

教程：<br>
	一般情况，直接以 回复 的方式回复bot转发过来的消息即可<br>
	特殊情况：<br>
		当你收到一条bot转发的图文消息时候，不能直接以 回复 的方式回复bot<br>
		而是需要以 "(UIN)内容" 的形式回复消息(见图一)<br>
